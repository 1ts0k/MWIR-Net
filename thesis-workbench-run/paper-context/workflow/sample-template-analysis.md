# 样文与规范分析

学校规范PDF要点：

- 中文题目一般不超过25个汉字。
- 中文摘要300-600字，英文摘要约300个英文实词。
- 关键词3-5个。
- 毕业论文正文字数一般不少于12000字。
- 参考文献按正文首次引用顺序编号，学校PDF一般要求不少于15篇，任务书提高为不少于20篇。
- 页面为A4，上2.5cm、下2cm、左2.5cm、右2cm，正文宋体小四号，固定值20磅。
- 图题置于图下，表题置于表上，均按章编号。

样文DOCX观察：

- 样文采用“第1章”“1.1”“1.1.1”层级。
- 图表主要集中在理论模型、实验实现和结果分析章节。
- 样文仅作为结构与节奏参考，不作为格式优先级来源。
