# Visual Comparisons

This directory stores thesis-ready MWIR-Net visual comparison images.

| File | Content |
|---|---|
| `rain100l_comparison.png` | Rain100L deraining comparison for samples `1`, `10`, `25`, `50`, and `100`. Columns: Input, PromptIR Official, PromptIR Init, MWIR-Net Init, MWIR-Net TTA, GT. |
| `sots_outdoor_comparison.png` | SOTS outdoor dehazing comparison for samples `0001_0.8_0.2`, `0030_0.95_0.12`, `0066_1_0.08`, `0100_0.9_0.12`, and `0056_0.8_0.16`. Columns: Input, PromptIR Official, PromptIR Init, MWIR-Net Init, MWIR-Net TTA, GT. |
| `ppt_5k_protocol/` | PPT-ready grids and single-sample strips generated from the 5k+5k protocol. |

The images are generated by `tools/make_visual_comparisons.py` and `tools/make_ppt_visual_comparisons.py`.
