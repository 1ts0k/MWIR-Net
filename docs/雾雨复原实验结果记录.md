# 雾雨图像复原实验结果记录

记录日期：2026-04-28

## 实验目标

本实验围绕本人设计并实现的 MWIR-Net 展开，验证模型在雨纹去除和室外去雾任务中的复原能力。MWIR-Net 采用多尺度 Transformer 编码解码结构，在解码阶段引入天气提示模块和通道注意力机制，并在训练中使用 Charbonnier 重建损失与 Sobel 边缘一致性约束。最终主模型采用公开复原权重初始化后进行联合微调，再进行二阶段去雨微调。论文中应明确该实验设置，不能表述为完整从零训练达到官方指标。

最终使用的 checkpoint 为：

```bash
checkpoints/mwirnet_ckpt_stage2_charb_edge002/epoch=1-step=856.ckpt
```

## 训练与测试设置

| 阶段 | 关键设置 | 说明 |
|---|---|---|
| 联合微调 | `derain + dehaze`，12 epoch，batch size 32，patch size 128 | 训练样本为 5000 张去雨图像和 5000 张去雾图像 |
| 权重初始化 | `PromptIR/ckpt/model.ckpt` | 只加载形状兼容的公开复原权重 |
| 二阶段去雨微调 | `derain`，8 epoch，学习率 `1e-5` | 从联合微调权重继续训练 |
| 最终损失 | Charbonnier + `edge_loss_weight=0.02` | 兼顾像素重建与边缘一致性 |
| 普通推理 | 单 checkpoint 直接测试 | 用于多测试集结果统计 |
| TTA 推理 | 8 向旋转和翻转自集成 | 用于最终主结果展示 |

## 主实验结果

| 方法 | 训练或推理设置 | Rain100L PSNR / SSIM / LPIPS | SOTS outdoor PSNR / SSIM / LPIPS |
|---|---|---:|---:|
| PromptIR 官方权重 | 官方预训练权重直接测试 | 37.44 / 0.9786 / 0.016323 | 30.59 / 0.9779 / 0.013528 |
| AirNet | 从零训练，5k 去雨 + 5k 去雾，12 epoch | 23.31 / 0.7928 / 0.267186 | 23.74 / 0.9309 / 0.054032 |
| PromptIR | 从零训练，5k 去雨 + 5k 去雾，12 epoch | 24.19 / 0.7971 / 0.259113 | 26.59 / 0.9437 / 0.029323 |
| PromptIR | 官方权重初始化微调，5k 去雨 + 5k 去雾，12 epoch | 31.63 / 0.9377 / 0.090232 | 30.75 / 0.9792 / 0.012616 |
| MWIR-Net | 从零训练，5k 去雨 + 5k 去雾，12 epoch | 25.16 / 0.8126 / 0.259754 | 26.22 / 0.9464 / 0.029608 |
| MWIR-Net | 官方权重初始化微调，5k 去雨 + 5k 去雾，12 epoch | 32.64 / 0.9387 / 0.091427 | 30.43 / 0.9780 / 0.012968 |
| MWIR-Net | L1 + edge0.03 二阶段微调 | 32.80 / 0.9397 / 0.094877 | 31.70 / 0.9801 / 0.010772 |
| MWIR-Net | Charbonnier + edge0.02 普通推理 | 32.89 / 0.9399 / 0.091470 | 31.72 / 0.9793 / 0.010678 |
| MWIR-Net | Charbonnier + edge0.02 + TTA | 33.33 / 0.9447 / 0.087578 | 32.07 / 0.9805 / 0.009871 |

从表中可以看出，在相同小样本微调设置下，AirNet 从零短训结果明显低于 PromptIR 和 MWIR-Net，说明其退化感知对比学习机制对训练时长和数据规模更敏感。MWIR-Net 初始化微调后的 Rain100L 指标高于 PromptIR 初始化微调结果。采用 Charbonnier 损失和 edge0.02 边缘一致性约束后，普通推理结果继续提升。TTA 自集成在 Rain100L 和 SOTS outdoor 上均带来增益，因此最终主结果采用单 checkpoint TTA 推理。

## 多测试集验证

### 去雨测试结果

| 测试集 | 图像数量 | PSNR | SSIM | LPIPS |
|---|---:|---:|---:|---:|
| Rain100L | 100 | 32.89 | 0.9399 | 0.091470 |
| Rain100H | 100 | 24.96 | 0.7799 | 0.239191 |
| Test100 | 98 | 23.81 | 0.8015 | 0.165610 |
| Test1200 | 1200 | 29.82 | 0.8696 | 0.090416 |
| Test2800 | 2800 | 30.57 | 0.9076 | 0.057636 |

Rain100L 和 Test2800 上结果较好，说明模型对轻中度雨纹和较规则雨纹具有较强复原能力。Rain100H 和 Test100 的指标相对较低，主要原因是雨纹密度更高、背景纹理更复杂，模型容易在雨纹去除和纹理保留之间产生折中。

### 去雾测试结果

| 测试集 | 图像数量 | PSNR | SSIM | LPIPS |
|---|---:|---:|---:|---:|
| SOTS outdoor | 500 | 31.72 | 0.9793 | 0.010678 |
| nyuhaze500 | 500 | 17.10 | 0.8223 | 0.101394 |

SOTS outdoor 与训练数据分布更接近，模型能够恢复较好的颜色和对比度。nyuhaze500 的 PSNR 和 SSIM 明显降低，说明跨数据集去雾仍存在泛化压力。该结果在论文中可作为模型局限讨论的依据。

## 消融实验

### 损失函数消融

| 损失设置 | Rain100L PSNR | Rain100L SSIM | Rain100L LPIPS |
|---|---:|---:|---:|
| L1 + edge0.01 | 32.47 | 0.9368 | 0.093612 |
| L1 + edge0.02 | 32.77 | 0.9380 | 0.094827 |
| L1 + edge0.03 | 32.80 | 0.9397 | 0.094877 |
| Charbonnier + edge0.02 | 32.89 | 0.9399 | 0.091470 |

Charbonnier + edge0.02 在 PSNR、SSIM 和 LPIPS 上整体更均衡，因此作为普通推理阶段的推荐损失设置。

### 模块消融

模块消融使用同一 checkpoint 进行推理诊断。该实验不重新训练各消融模型，主要用于观察天气提示和通道注意力在推理阶段对输出的影响。

| 模式 | 含义 | Rain100L PSNR | Rain100L SSIM | Rain100L LPIPS |
|---|---|---:|---:|---:|
| `full` | 完整 MWIR-Net | 32.89 | 0.9399 | 0.091470 |
| `zero_prompt` | 天气提示特征置零 | 32.85 | 0.9393 | 0.092309 |
| `no_channel_attention` | 关闭提示通道注意力 | 32.99 | 0.9415 | 0.088853 |

天气提示置零后指标略有下降，说明提示分支对模型有正向作用。关闭提示通道注意力后指标未下降，说明当前通道注意力在该 checkpoint 上的贡献不稳定。论文中不宜把该模块效果夸大，可表述为“提示特征对结果有一定影响，通道注意力仍需要通过重新训练消融模型进一步验证”。

### 推理增强消融

| 推理方式 | Rain100L PSNR / SSIM / LPIPS | SOTS outdoor PSNR / SSIM / LPIPS | 是否采用 |
|---|---:|---:|---|
| 普通单 checkpoint | 32.89 / 0.9399 / 0.091470 | 31.72 / 0.9793 / 0.010678 | 普通推理基线 |
| TTA 单 checkpoint | 33.33 / 0.9447 / 0.087578 | 32.07 / 0.9805 / 0.009871 | 是 |
| 3 checkpoint ensemble | 32.95 / 0.9402 / 0.094919 | 30.90 / 0.9738 / 0.012469 | 否 |
| 3 checkpoint ensemble + TTA | 33.30 / 0.9444 / 0.091291 | 31.17 / 0.9750 / 0.011790 | 否 |

checkpoint ensemble 对 Rain100L 的 PSNR 和 SSIM 只有小幅变化，LPIPS 反而变差，在 SOTS outdoor 上下降更明显。因此最终采用单 checkpoint TTA，而不采用 checkpoint ensemble。

## 传统弱基线

| 方法 | 测试集 | PSNR | SSIM | 输出目录 |
|---|---|---:|---:|---|
| 3×3 median filtering | Rain100L | 24.38 | 0.7327 | `traditional_output/median_rain100l` |
| CLAHE luminance enhancement | SOTS outdoor | 16.39 | 0.8131 | `traditional_output/clahe_sots_outdoor` |

传统弱基线的指标明显低于 MWIR-Net，说明单纯滤波或增强难以稳定处理雾雨退化。论文中可把这类方法作为本地复现的传统对照，不应作为前沿方法的强对比。

## 真实雨场景补充

已使用 `tools/prepare_gtrain_test.py` 整理 GT-RAIN 测试集。该划分包含 7 个真实雨场景、2100 张雨图，每张雨图对应同一场景的清晰参考帧。普通推理结果如下：

| 测试集 | 图像数量 | PSNR | SSIM | 说明 |
|---|---:|---:|---:|---|
| GT-RAIN-test | 2100 | 20.94 | 0.5960 | 仅作为参考，论文中主要用于主观分析 |

GT-RAIN 的参考清晰帧与雨帧并非严格逐像素配准，因此 PSNR 和 SSIM 只能作为辅助信息。论文中更稳妥的写法是选取若干真实雨图像展示输入、复原结果和参考帧，并分析模型对真实雨纹、局部反光和背景纹理的处理效果。

## 论文表述口径

可以写：

> 本文面向雾雨退化图像复原任务设计并实现了 MWIR-Net。该网络采用多尺度 Transformer 编码解码结构，在解码阶段引入天气提示模块和通道注意力机制，并结合 Charbonnier 重建损失与边缘一致性约束改善复原细节。实验在 RAIN13K、OTS、Rain100L、Rain100H、Test100、Test1200、Test2800、SOTS outdoor 和 nyuhaze500 等数据集上进行，评价指标包括 PSNR、SSIM 和 LPIPS。

可以写：

> 考虑到完整训练成本较高，本文采用公开复原权重初始化，并在去雨与去雾子集上进行联合微调。该设置用于验证所设计模块和训练策略在有限训练资源下的有效性，相关结果不表述为完整从零训练结果。

不建议写：

> 本文从零训练得到上述官方级指标。

不建议写：

> 本文已经在所有雾雨退化类型上完成全面验证。

当前实验主体覆盖雨纹去除、室外去雾和真实雨主观分析。非均匀雾、雨滴积聚、雨雾同时退化等复杂场景仍属于后续扩展方向。
