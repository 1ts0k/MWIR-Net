# 所有模型指标汇总

统计日期：2026-04-28

本文汇总当前工作区已经生成的模型输出结果。所有 PSNR、SSIM 和 LPIPS 均基于已保存的输出图像统一复算，复算时将预测图与目标图按共同尺寸做中心裁剪，并裁剪到 16 的整数倍。PSNR、SSIM 越高越好，LPIPS(alex) 越低越好。

## 口径与来源

- 输出目录扫描范围：`MWIR-Net/outputs`、`PromptIR/official_output`、`PromptIR/promptir_output_5k_12epoch_init`、`MWIR-Net/traditional_output`。
- 目标图像目录：`MWIR-Net/test/derain/*/target` 与 `MWIR-Net/test/dehaze/*/target`。
- 去雾文件匹配方式：预测文件名按第一个下划线前的 clean id 匹配目标图，例如 `1400_1.png` 匹配 `1400.png`。
- 去雨文件匹配方式：优先使用同名目标图；当后缀不同，如 Test2800 的 `.jpg` target 与 `.png` 输出，按文件 stem 匹配。
- LPIPS 使用 `alex` backbone；其数值越低代表感知距离越小。
