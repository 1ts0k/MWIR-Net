## 训练协议

为避免不同输出目录之间的训练设置混淆，本文采用如下协议描述和命名口径。

### 数据与划分

- 去雨训练数据来自 RAIN13K 训练集，脚本通过 `tools/prepare_mwir_data.py` 生成 `data/Train/Derain/rainy` 与 `data/Train/Derain/gt` 软链接，并写入 `data_dir/rainy/rainTrain.txt`。
- 去雾训练数据来自 RESIDE/OTS 或 ITS 目录，当前主要实验使用 `python tools/prepare_mwir_data.py --dehaze-source ots` 生成 `data/Train/Dehaze/synthetic` 与 `data/Train/Dehaze/original` 软链接，并写入 `data_dir/hazy/hazy_outside.txt`。
- 去雨测试集包括 Rain100L、Rain100H、Test100、Test1200、Test2800 和 GT-RAIN-test；去雾测试集包括 SOTS outdoor 与 nyuhaze500。
- 5k 协议表示从去雨和去雾训练样本中各取最多 5000 张，使用 `--max_derain 5000 --max_dehaze 5000` 控制；若进行单任务二阶段微调，则只启用对应任务的样本上限。

### 训练设置

| 阶段/版本 | 任务 | 初始化 | 主要训练参数 | 损失设置 | 用途 |
|---|---|---|---|---|---|
| MWIR-Net-5k_12epoch | 去雨+去雾 | 随机初始化 | `epochs=12`、`batch_size=32`、`patch_size=128`、`lr=2e-4`、`warmup_epochs=2`、`precision=16-mixed` | L1 + `0.05` Sobel 边缘一致性 | 从零训练基线 |
| MWIR-Net-5k_12epoch_init | 去雨+去雾 | 兼容公开复原权重初始化 | 同 5k 协议 | L1 + `0.05` Sobel 边缘一致性 | 主干初始化对比 |
| MWIR-Net-stage2_charb_edge002 | 去雨二阶段 | `MWIR-Net-5k_12epoch_init` 最后一轮权重 | `epochs=8`、`lr=1e-5`、`warmup_epochs=1`、`max_derain=5000` | Charbonnier(`eps=1e-3`) + `0.02` Sobel 边缘一致性 | 主要去雨模型 |
| MWIR-Net-dehaze_stage2_charb_edge002 | 去雾二阶段 | 兼容阶段权重初始化 | `epochs=4`、去雾单任务训练 | Charbonnier + `0.02` Sobel 边缘一致性 | 去雾二阶段 sweep |
| Fair-ablation-full | 去雨 | `MWIR-Net-5k_12epoch_init` 最后一轮权重 | `epochs=2`、`lr=1e-5`、`max_derain=5000`、`ablation_mode=full` | Charbonnier + `0.02` Sobel | 公平消融基准 |
| Fair-ablation-zero_prompt | 去雨 | 同上 | `epochs=2`、其余参数同 Fair-ablation-full，仅设置 `ablation_mode=zero_prompt` | Charbonnier + `0.02` Sobel | 验证 prompt 分支影响 |
| Fair-ablation-no_channel_attention | 去雨 | 同上 | `epochs=2`、其余参数同 Fair-ablation-full，仅设置 `ablation_mode=no_channel_attention` | Charbonnier + `0.02` Sobel | 验证 prompt 通道注意力影响 |

训练优化器统一为 AdamW，学习率调度器为线性 warmup + cosine annealing。所有 checkpoint、日志和输出图像只保存在本地目录中，不纳入 Git 版本管理。

### 推理与复算

- 常规推理使用 `test.py`，单张输入 batch size 为 1，输出目录按任务和 split 命名。
- TTA 使用 8 路旋转/翻转自集成，即 4 个旋转角度及其水平翻转结果，恢复后做均值融合。
- checkpoint ensemble 使用多个 checkpoint 输出均值融合；当前结果中 ensemble 作为补充对比，不作为最终推荐方案。
- PSNR、SSIM 和 LPIPS 均以保存后的输出图像为准统一复算，复算时预测图和目标图按共同尺寸中心裁剪，并裁剪到 16 的整数倍。

## 口径与来源

- 输出目录扫描范围：`MWIR-Net/outputs`、`MWIR-Net/outputs/fair_ablation_*`、`PromptIR/official_output`、`PromptIR/promptir_output_5k_12epoch_init`、`MWIR-Net/traditional_output`。
- 目标图像目录：`MWIR-Net/test/derain/*/target` 与 `MWIR-Net/test/dehaze/*/target`。
- 去雾文件匹配方式：预测文件名按第一个下划线前的 clean id 匹配目标图，例如 `1400_1.png` 匹配 `1400.png`。
- 去雨文件匹配方式：优先使用同名目标图；当后缀不同，如 Test2800 的 `.jpg` target 与 `.png` 输出，按文件 stem 匹配。
- LPIPS 使用 `alex` backbone；其数值越低代表感知距离越小。
